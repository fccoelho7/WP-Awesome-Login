Change the way you log in WordPress.

== Description ==

If you want to change the login form style, this plugin is for you!

= Usage =

1. Download and extract `wp-awesome-login.zip` to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Configurations > WP Awesome Login
