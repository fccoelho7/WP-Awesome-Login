Change the way you log in to WordPress with a flat design and colors of your choice.

== Description ==
Are you tired of the standard login screen WordPress? You want to leave her with the face of your website? this is the plugin you were looking for, he does all that and a little more.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/wp-awesome-login` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the \'Plugins\' screen in WordPress
1. Use the Apparence->WP Awesome Login screen to configure the plugin
